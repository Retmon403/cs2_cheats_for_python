# 问题
你是不是想问我，运行起来为什么这么卡？  
答：用易语言就不会卡  

为什么不用c++，却用python？  
答：因为python写起来简单，代码量比c++少一万倍  

如何更新offset?  
答：main.py中 变量->UPDATE_OFFSET默认为TRUE 程序会自动更新偏移量
